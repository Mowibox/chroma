Bonjour à tous ! Comme promis à la création du site, **Chroma s'ouvre officiellement à la contribution !**

Que vous souhaitiez partager vos idées, proposer des améliorations ou alimenter le site avec vos propres tutoriels, toute aide est la bienvenue.

## Comment contribuer ?

* **:speech_balloon: Partagez vos suggestions sur Discord :** utilisez le channel [Review and Feedback](https://discord.com/channels/1383790297594728518/1386648737366937701) pour poser des questions, suggérer des améliorations ou discuter d'idées avec d'autres contributeurs.
  
* **:warning:  Signaler des problèmes sur GitHub :** Vous avez trouvé un bug ? Créez une [issue sur le GitHub de Chroma](https://github.com/Mowibox/chroma/issues/new/choose) afin qu'elle puisse être suivie et corrigée.
  
* **:pencil: Rédigez des tutoriels ou contribuez au développement du site web :** Contribuez à la rédaction de nouveaux tutoriels, améliorez ceux qui existent déjà ou plongez-vous dans le code source pour améliorer le site.

## Le guide complet

Le guide détaillé pour savoir comment contribuer est disponible à cette adresse : [Comment contribuer à Chroma ?](https://mowibox.github.io/chroma/fr/tutorials/miscellaneous/how_to_contribute/)
